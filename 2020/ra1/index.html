Baire 纲定理的各种等价形式

不得不说各种书用的各种定义各种等价形式整的人头疼…但无论如何, 我们都需要从下面的定义出发.

无处稠密集 (疏集): 一个在全空间的任何开集中都不稠密的集合称为无处稠密集. 可以验证这当且仅当集合的闭包没有内点.

开稠集: 开的稠密集. 一个集合是开稠集显然等价于其补为无处稠密集.

第一纲集: 疏集的可数并称为第一纲集.

第二纲集: 不是第一纲集的集合称为第二纲集. (这名字也太简陋了)

Baire 纲定理: (1) 完备度量空间是第二纲的.

(2) 一列开稠集之可数交仍为开稠集.

(3) 一列没有内点的闭集之并仍没有内点.

这些叙述都是等价的, 证明略去.

题目

  1. 证明 $C[0,1]$ 中处处不可微的函数全体组成的集合在 $C[0,1]$ 中为第二纲集.
  2. 证明无穷维 Banach 空间的 Hamel 基不可数.
  3. 设 $f:\mathbb{R}^n\to \mathbb{R}^m$ 满足 Stepanov 型条件:$\limsup\limits_{y\to x} \dfrac{|f(y)-f(x)|}{y-x}<\infty$ 对任意 $x\in \mathbb{R}^n$ 成立. 证明存在开集 $U\subset \mathbb{R}^n$ 使得 $f|_U$ 为 Lipschitz 函数.
  4. Let $f\in C^\infty(\mathbb{R})$ be a function with the following property: $\forall\ x\in \mathbb{R}$, exists positive integer $n(x)$ such that $f^{(n(x))}(x)=0$. Then $f$ is a polynomial.

更多习题可参考 C. Costara, D. Popa, Exercises in Functional Analysis.

解答/思路

  1. 太多了逃了

  2. 假设是可数维的 (这种说法不严谨), 不妨设为 $X=\text{span}(x_i)_{i\in \mathbb{N}}$, 令 $X_n=\text{span}(x_i)_{i=1}^n$, 则 $X_n$ 为闭集, 并且 $\bigcup_{n\in \mathbb{N}} X_n=X$. 于是由 Baire 纲定理, 只要证 $X_n$ 都没有内点.

  3. 还没做, 后面补上

  4. 用两次 Baire 纲定理.
